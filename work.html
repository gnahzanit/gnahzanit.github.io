Onionshop is a drawing program that has a canvas which takes mouse or tablet input and outputs pixel line strokes (scalar art). Using different tools within Onionshop, one can draw or erase lines and shapes, undo redo strokes, and manage color palettes. Drawings can be saved to files with the extension .onion, and reopened later to continue editing.

A snippet of the project running on a small sample of the rehydrated Harvard Dataverse: GWU Libraries Dataverse: Climate Change Tweets Ids.

The Climate Change Twitter Map project aimed to analyze public engagement and perspectives on climate change through tweets. By examining which climate-related topics and events gained traction on Twitter, the project sought to identify both popular and overlooked issues. The goal was to understand global sentiment and conversation trends about climate change. The project utilized filtering and aggregation of Twitter data by location, date, and keywords, and employed Plotly for visual representation. The final interactive map allowed users to explore topic popularity over time, offering insights into how climate change discussions vary across different regions and events.

Phisherman

Threat management tool

Hackathon
Aug 2020

PROJECT OVERVIEW

ðŸ“Ž Project Page

Project video with the Phisherman logos and opening animation by me.

PhisherMan is a phishing threat management tool that receives reports of potential phishing threats, quickly filters through spam and automatically files malicious pages in its internal database for further action. When PhisherMan receives a suspicious email, it uses a machine learning algorithm to determine if the email is indeed a phishing threat or spam. If the system determines the email as a phishing threat, it creates and stores it in a database to be handled and mitigated. This saves a lot of time performing manual labor so that serious threats can be identified and addressed as soon as possible.

Sprout

UX Project for Women In STEM

University Project
MAY 2023 - AUG 2023

PROJECT OVERVIEW

ðŸ“Ž Formative Studies ðŸ“Ž Design Concepts and Prototypes ðŸ“Ž Project Results Report

The Sprout project was created in response to the fact that while 34% of STEM bachelor's degree holders are women, they make up only 23% of Canadians working in science and technology. In the male-dominated STEM field, women often face workplace challenges that can lead to their withdrawal from these careers. Recognizing this, the Sprout project focuses on creating spaces where women can build personal connections and receive mentor-mentee guidance, instilling confidence in those pursuing or already working in STEM. Sprout is designed as a user-centered prototype aimed at addressing the unique challenges women face in the STEM experience.
