ONIONSHOP

A scalar art application

University Project
Sept 2021 - Dec 2021

PROJECT OVERVIEW

üìé Project Specification üìé Project Repository

Onionshop is a drawing program that has a canvas which takes mouse or tablet input and outputs pixel line strokes (scalar art). Using different tools within Onionshop, one can draw or erase lines and shapes, undo redo strokes, and manage color palettes. Drawings can be saved to files with the extension .onion, and reopened later to continue editing.

Sprout

UX Project for Women In STEM

University Project
MAY 2023 - AUG 2023

PROJECT OVERVIEW

üìé Formative Studies üìé Design Concepts and Prototypes üìé Project Results Report

The Sprout project was created in response to the fact that while 34% of STEM bachelor's degree holders are women, they make up only 23% of Canadians working in science and technology. In the male-dominated STEM field, women often face workplace challenges that can lead to their withdrawal from these careers. Recognizing this, the Sprout project focuses on creating spaces where women can build personal connections and receive mentor-mentee guidance, instilling confidence in those pursuing or already working in STEM. Sprout is designed as a user-centered prototype aimed at addressing the unique challenges women face in the STEM experience.
